# FINTECH

Este repositorio contiene una colección de implementaciones enfocadas en el uso de Retrieval-Augmented Generation (RAG) aplicadas a problemas del sector financiero, específicamente relacionados con empresas del S&P 500.

El objetivo principal fue desarrollar soluciones que permitan consultar, analizar y generar insights financieros a partir de grandes volúmenes de información, combinando modelos de lenguaje con técnicas de recuperación de datos relevantes.

🔍 ¿Qué incluye este proyecto?

Implementación de pipelines RAG para:
- Consulta inteligente de información financiera
- Análisis de tendencias del mercado
- Exploración de datos de empresas del S&P 500
- Integración de embeddings para mejorar la recuperación semántica
- Uso de modelos de lenguaje para generación de respuestas contextualizadas
- Procesamiento y estructuración de datos financieros

⚙️ Tecnologías utilizadas
- Python
- Modelos de lenguaje (LLMs)
- Embeddings y vector stores
- Frameworks de RAG
